Quick Start Guide

Follow these steps to connect a Raspberry Pi, Intel NUC, or any Linux device to your signage system.

1. Register Device

Add your device using the "Add Device" button with a unique Device ID.

2. Download Client
Download Client Script
3. Install Dependencies

On your device, install Python and a media player.

Quick Setup Commands

Run these on your Raspberry Pi or Linux device:

# Install dependencies
sudo apt install python3 python3-pip omxplayer -y
pip3 install requests

# Download and setup
mkdir ~/signage && cd ~/signage
wget {{ request.url_root }}download/client
chmod +x signage_client.py

# Configure (edit with your values)
export SIGNAGE_SERVER_URL={{ request.url_root.rstrip('/') }}
export DEVICE_ID=your-device-id

# Test run
python3 signage_client.py
Supported Devices & Players
Raspberry Pi
  • omxplayer (recommended)
  • VLC (alternative)
Intel NUC / PC
  • VLC (recommended)
  • ffplay (lightweight)
Any Linux Device
  • Automatic detection
  • Fallback support
Important: Make sure your device can reach this server at {{ request.url_root }} and that the Device ID matches exactly what you register in the system.
